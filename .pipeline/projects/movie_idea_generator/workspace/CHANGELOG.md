# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [v0.1.0] — 2025-01-01

### Added
- Core movie idea generation engine with rule-based template system
- Support for 10 genres: Action, Comedy, Drama, Horror, Sci-Fi, Romance, Thriller, Fantasy, Mystery, Adventure
- CLI entry point (`movie-idea-generator`) with flags for genre, count, format, and seed
- Programmatic API via `MovieIdeaGenerator` class
- Text and JSON output formats
- Character generation with randomized names, roles, and descriptions
- Input validation with descriptive error messages
- Comprehensive test suite (68+ tests)
- Modern Python packaging with pyproject.toml
- MIT License
- Deployment documentation
