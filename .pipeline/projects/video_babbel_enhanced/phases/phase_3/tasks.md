# Phase 3 Tasks

- [ ] Task 1: Flask app skeleton + upload + REST API layer
  - What: Create the Flask web app with file upload handling, clip listing, and REST API endpoints. Wire upload to the existing extraction pipeline (transcribe → translate → score → extract).
  - Files:
    - `workspace/app.py` — Flask app with routes: `/upload`, `/clips`, `/extract`, `/stats`
    - `workspace/templates/upload.html` — HTML upload page with drag-and-drop, source/target language selects, N-clips input
    - `workspace/templates/clips.html` — HTML clip listing page with links to watch/drill
    - `workspace/static/css/style.css` — basic styling
  - Done when:
    - `python app.py` starts Flask on localhost:5000
    - Upload page accepts a video file, source/target language, and N clips
    - Submitting the upload form triggers the extraction pipeline (calls `cmd_extract` from `cli.py`)
    - `GET /clips` returns JSON list of all clips from the database
    - `GET /stats` returns JSON with total/due/new/mastered counts from `get_session_stats`
    - Unit tests in `tests/test_app.py` verify: upload endpoint saves file, extract endpoint calls pipeline, /clips returns correct data, /stats returns valid counts
